# Ficha de Cadastro de Novo Colaborador — Grupo R7

Formulário em HTML para preenchimento dos dados de admissão de colaboradores, no padrão visual do Grupo R7 Serviços Contábeis e Administrativos.

## O que é

Um arquivo único (`ficha_cadastro.html`) que funciona como formulário digital. Não precisa de instalação, servidor ou internet para funcionar — basta abrir no navegador.

## Como usar

1. **Abrir o arquivo**: dê duplo clique em `ficha_cadastro.html` (ou envie o link, se estiver hospedado em um site) e ele abrirá em qualquer navegador (Chrome, Edge, Firefox, Safari).
2. **Fechar o aviso inicial**: ao abrir, aparece um aviso informando que nenhum dado é salvo. Clique em "Entendi, continuar" para acessar o formulário.
3. **Preencher os campos**: dados da empresa, dados pessoais, endereço, dependentes e documentos. Campos com `*` são obrigatórios.
4. **Imprimir ou salvar em PDF**: clique em "Imprimir". O navegador abrirá a opção de impressão — escolha uma impressora física ou "Salvar como PDF" para gerar o arquivo.
5. **Entregar ao Departamento Pessoal**: envie o PDF gerado ou a impressão física para o escritório, pelos canais de contato informados no rodapé do formulário.

## ⚠️ Importante: nenhum dado é armazenado

Este formulário **não envia, não salva e não transmite** nenhuma informação para o Grupo R7 ou para qualquer servidor. Todo o preenchimento acontece apenas na memória do navegador, no computador do próprio usuário.

- Não há banco de dados, planilha ou e-mail automático recebendo essas informações.
- **Ao fechar a aba, atualizar a página ou clicar em "Limpar", todos os dados digitados são perdidos permanentemente.**
- Por isso, é essencial gerar o PDF (ou imprimir) **antes** de fechar a página, e depois enviar esse arquivo/impresso ao Departamento Pessoal por um canal seguro.

## Onde hospedar (para enviar um link aos clientes)

Como é um arquivo estático, pode ser hospedado gratuitamente em serviços como Netlify Drop ou GitHub Pages, ou no site do próprio escritório, gerando um link fixo para os clientes acessarem e preencherem diretamente no navegador.

## Estrutura de dados sensíveis (LGPD)

O formulário coleta dados pessoais sensíveis (CPF, RG, salário, dados de dependentes menores). Como nada é armazenado eletronicamente pelo próprio formulário, a responsabilidade pelo tratamento desses dados começa no momento em que o PDF/impresso é entregue ao Departamento Pessoal — a partir daí, seguem as políticas internas de proteção de dados do Grupo R7.
